# ProConnect

ProConnect is a professional networking platform built using the MERN stack.  
Users can create profiles, connect with others, and share posts.

## 🚀 Features
- User authentication
- Profile creation and management
- Create and share posts
- Responsive UI
- Secure backend API

## 🛠 Tech Stack
Frontend
- React
- Tailwind CSS
- Vite

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
