# ProConnect

ProConnect is a **professional networking platform** built using the **MERN stack**.
Users can create profiles, connect with others, and share posts in a modern responsive interface.

---

## 🚀 Features

* User Authentication (Login / Signup)
* Profile Creation and Management
* Create and Share Posts
* Responsive UI
* RESTful API Backend
* Secure Authentication System

---

## 🛠 Tech Stack

### Frontend

* React
* Tailwind CSS
* Vite

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation

### 1️⃣ Clone the repository

git clone https://github.com/vivekmeena57/ProConnect_Project.git

### 2️⃣ Install backend dependencies

cd backend
npm install

### 3️⃣ Install frontend dependencies

cd frontend
npm install

### 4️⃣ Run backend

npm run dev

### 5️⃣ Run frontend

npm run dev
